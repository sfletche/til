# flexbox Cheatsheet
An excellent, more formalized cheatsheet can be found [here](https://yoksel.github.io/flex-cheatsheet/#display)

## Ordering

### flex-direction
* row
* row-reverse
* column
* column-reverse

### flex-wrap
* nowrap
* wrap
* wrap-reverse

### flex-flow
shorthand for `flex-direction` and `flex-wrap`
* row nowrap
* column-reverse
* column wrap
* row-reverse wrap-reverse

### order
* -1 
* 0 
* 1

* flex-end

### align-items

* flex-end

### align-self

* flex-end
